# MMM

The NetLogo source code is contained in the mmm.nls file. Open it in [NetLogo](https://ccl.northwestern.edu/netlogo/download.shtml)  
The NetTango block interface is contained in the blocks.ntjson file. Follow these [steps](https://github.com/Systems-Learning-and-Development-Lab/MMM/wiki/Open-ntjson-file-in-NetTango-editor) to open it in the NetTango editor and edit the blocks.
